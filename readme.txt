/********************************************************************************
* 工程详情
********************************************************************************/
1. 本工程为modbus通讯示例
2. 提供了最为常用的 rtu、tcp通讯
3. 提供了最为常用的功能码封装处理 03H、04H、06H、10H、01H、05H、0fH
4. 该工程精简掉其他modbus功能码的处理，并简化了处理流程，突出modbus主要处理
5. 该工程的 rtu、tcp报文处理的是共享同一套地址，这是支持modbus通讯产品的常见应用
